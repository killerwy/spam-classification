## 📝 项目简介
一套稳定易用的垃圾邮件分类系统，基于Python与机器学习算法构建。本应用配备图形化操作界面（GUI），支持用户加载邮件数据集、训练多种分类模型、评估模型效果，同时可对单封邮件进行检测，判断其为垃圾邮件或正常邮件。

系统内置多种机器学习算法，可输出完整模型评估指标，便于用户直观了解模型分类效果，既可用于学习教学，也可落地实现垃圾邮件识别场景。

## ✨ 功能特性
### 核心功能
1. **数据加载**：支持按目录结构读取邮件数据集（数据集需划分ham正常邮件、spam垃圾邮件子文件夹）
2. **模型训练**：
   - 内置朴素贝叶斯、支持向量机、随机森林三种分类算法
   - 自动按照8:2比例划分训练集与测试集，用于模型效果评估
3. **效果评估**：
   - 输出完整评估指标：准确率、精确率、召回率、F1分数
   - 生成完整分类报告
   - 格式化清晰展示全部评估结果
4. **邮件识别**：
   - 手动输入邮件正文并预测是否为垃圾邮件
   - 支持从文本文件导入邮件内容
   - 输出分类结果对应的概率分值

### 图形界面
- 分区清晰、操作简单的可视化界面
- 可滚动文本框用于邮件输入与评估结果展示
- 底部状态栏实时反馈程序运行状态
- 自定义字体与界面样式，提升阅读舒适度
- 异常捕获机制，弹窗友好提示错误信息

## 📖 使用教程
### 1. 加载数据集
- 点击「浏览...」按钮，选中邮件数据集根目录

> 推荐数据集：Enron 邮件数据集。官方来源（卡内基梅隆大学）：https://www.cs.cmu.edu/~enron/
>
> 雅典经济与商业大学版 Enron 标注垃圾邮件子集：https://www2.aueb.gr/users/ion/data/enron-spam/

数据集目录结构规范如下：
```
数据集/
├── 文件夹1/
│   ├── ham/      # 正常邮件（标签0）
│   └── spam/     # 垃圾邮件（标签1）
├── 文件夹2/      # 可选附加子文件夹
│   ├── ham/
│   └── spam/
├── ...
└── 文件夹n/      # 可选附加子文件夹
    ├── ham/
    └── spam/
```
- 点击「加载数据」处理数据集，界面会展示邮件总量统计信息

### 2. 训练分类模型
- 在下拉菜单选择分类器（默认朴素贝叶斯）
- 点击「训练模型」完成全部可用模型训练
- 在评估结果区域查看完整性能指标

### 3. 单封邮件分类检测
- 将邮件正文粘贴至「邮件分类」输入框（也可从文件导入）
- 点击「预测」获取分类结果
- 页面展示判定结果（垃圾邮件/正常邮件）与对应概率分值
- 点击「清空」重置输入框内容
